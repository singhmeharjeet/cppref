# C++ Package Manager
Conan is a C++ package manager that simplifies the process of managing dependencies in C++ projects.
It allows developers to easily find, install, and manage libraries and packages, making it easier to
build and maintain C++ applications.

## Installing Conan
To install Conan, you can use the following command:

```bash copy
pip3 install conan
```
